GitHub-OS Guide

Selamat datang di GitHub-OS. Ini adalah ekosistem Cloud Desktop pribadi yang menghubungkan GitHub sebagai frontend dan Google Drive sebagai storage backend.

🪟 Window Management

  • Multi-Tasking: Klik ikon Explorer di Start Menu untuk membuka jendela baru.
  • Aero Snap: Seret jendela ke tepi layar untuk otomatis membagi layar (Split Screen).
  • Minimize/Restore: Gunakan tombol _ untuk menyembunyikan jendela ke Taskbar.

⌨️ Keyboard Shortcuts

ShortcutAction
Alt + NBuka Explorer Baru

Alt + /Tampilkan Shortcut Hub

💻 Terminal Commands

Gunakan Terminal (Alt + T) untuk eksekusi cepat:

  • ls - Melihat daftar file/folder & ID uniknya.
  • mkdir [nama] - Membuat folder baru di Drive.
  • rm [id] - Memindahkan file ke Trash Bin.
  • quota - Melihat sisa kapasitas Google Drive asli.

🔒 Security & Privacy

Sistem ini menggunakan SHA-256 Hashing untuk login. Password tidak disimpan dalam bentuk teks biasa, sehingga aman dari intipan source code.

Lock Screen: Tekan Alt + L jika ingin meninggalkan komputer tanpa Logout.

📡 Technical Setup

  1. Pastikan GAS URL sudah terpasang di index.html.
  2. Pastikan file inner_explorer.html berada di satu folder repositori yang sama.
  3. Aktifkan GitHub Pages melalui Settings > Pages.
